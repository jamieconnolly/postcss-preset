> My own opinionated PostCSS preset.

## Installation

```
$ npm install --save-dev @jamieconnolly/postcss-preset
```

## Usage

To get started, add this to your `.postcssrc.js` file:

```js
module.exports = require('@jamieconnolly/postcss-preset');
```
